# Buiry MCP — GitHub Copilot Setup

## Current Status

GitHub Copilot does not yet support MCP servers natively. Buiry works with Copilot through the AI_Starter.md file.

## Manual Setup

1. Copy `AI_Starter.md` to your project root
2. Copy `Build-Context-Memory.json` to your project root
3. Start your Copilot session — the AI reads AI_Starter.md first
4. The AI updates Build-Context-Memory.json at the end of each session

## Future

When Copilot adds MCP support, the setup will be similar to Claude Code.
